# Social Media App Assignment

## Run Locallly

1. Fork the repo

2. Clone the forked repo

```bash copy
git clone https://github.com/<your_github_username>/social-media-assignment.git
```

3. Change directory

```bash copy
cd social-media-assignment
```

4. Install dependencies

```bash copy
npm run install
```

5. Run

```bash copy
npm run dev
```
